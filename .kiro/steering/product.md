# プロダクト概要

## 製品名
enecoQ Data Fetcher

## 概要
enecoQは株式会社ファミリーネットジャパンが提供するCYBERHOMEサービス内の電力データ管理Webサービスです。このツールは、enecoQから電力データをプログラマティックに取得し、JSON形式やコンソール表示で出力するCLIツールです。

## 目的
enecoQ Webサービスから電力使用量、電力使用料金、CO2排出量データを自動的に取得し、データ分析や自動化を可能にする

## 主な機能
- enecoQ Web Serviceへの安全な認証
- 電力使用量データの取得（kWh単位）
- 電力使用料金データの取得（円単位）
- CO2排出量データの取得（kg単位）
- 今日または今月のデータ取得
- JSON形式またはコンソール表示での出力
- 詳細なエラーハンドリングとログ機能
- 設定ファイルによるカスタマイズ

## ターゲットユーザー
- CYBERHOME（enecoQ）のアカウントを持つユーザー
- 電力データを自動的に取得・分析したいユーザー
- 電力使用量の可視化や分析を行いたい開発者
- エネルギー管理システムと連携したいユーザー

## 必要要件
- CYBERHOME（enecoQ）のアカウント
- Python 3.9以上
- uv（Pythonパッケージマネージャー）
- Playwright（ブラウザ自動化）

## 出力形式

### JSON形式
```json
{
  "period": "month",
  "timestamp": "2024-01-15T10:30:00",
  "usage": 250.5,
  "cost": 7515.0,
  "co2": 125.25
}
```

単位:
- `usage`: kWh（キロワット時）
- `cost`: JPY（日本円）
- `co2`: kg（キログラム）

### コンソール形式
```
==================================================
enecoQ Power Data
==================================================

Period: month
Timestamp: 2024-01-15 10:30:00

Power Usage: 250.5 kWh
Power Cost: 7515.0 JPY
CO2 Emission: 125.25 kg

==================================================
```

## ログ機能
- **コンソール**: INFOレベル以上
- **ファイル**: `logs/enecoq.log`（DEBUGレベル以上）
- 認証情報は記録されない（セキュリティ保護）
